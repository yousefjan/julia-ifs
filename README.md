3D Reversed Julia Set IFS (Legacy + Rust Port)

- Legacy C/C++ sources in `legacy/` are public domain (see `legacy/Legal.txt`).
- A Rust port is being developed in `rust/` using a software renderer.

Build and run Rust preview:

```bash
cd rust && cargo run --release
```
